# Codigos PlantUML - Finesse Sportz

Esta pasta contem os diagramas principais do projeto em PlantUML.

## Lista de Diagramas

| Arquivo | Finalidade |
|---|---|
| `actors.puml` | Atores e sistemas externos. |
| `use_cases.puml` | Casos de uso principais. |
| `architecture.puml` | Arquitetura geral do sistema. |
| `components.puml` | Componentes e microsservicos. |
| `classes.puml` | Entidades e relacionamentos de dominio. |
| `diagrama_comunicacao_fluxo_completo.puml` | Comunicacao completa do fluxo de compra. |
| `diagrama_sequencia_pedido_pagamento.puml` | Sequencia detalhada de pedido e pagamento. |
| `estados.puml` | Estados do pedido. |
| `diagrama_entidade_relacionamento.puml` | Modelo de dados relacional. |
| `implantacao.puml` | Implantacao planejada. |

## Como Visualizar

Use uma extensao PlantUML no VS Code ou abra os arquivos no site oficial:

https://plantuml.com
